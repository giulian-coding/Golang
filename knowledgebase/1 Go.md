# Golang

Go ist eine statisch typisierte Programmiersprache, die von Google entwickelt wurde. Sie wurde entwickelt, um die Produktivität von Entwicklern zu erhöhen und gleichzeitig die Leistung und Effizienz von Software zu verbessern. Go bietet eine einfache Syntax, schnelle Kompilierung und eine starke Unterstützung für Nebenläufigkeit.

Der Compiler von Go verwendet eine klare und konsistente Syntax, die es Entwicklern ermöglicht, schnell und effizient Code zu schreiben. In Go steht der Name einer Variable immer links und der Typ immer rechts, was die Lesbarkeit verbessert und Fehlerquellen reduziert. Er kompliert direkt zu Maschinencode, was zu einer schnellen Ausführung führt. Go bietet auch eine eingebaute Unterstützung für Nebenläufigkeit durch Goroutinen und Kanäle, was die Entwicklung von hochleistungsfähigen Anwendungen erleichtert.

## Urspünge von Go

- Google schafft mit Go 2007 eine Alternative zu C++ und Java, um die Produktivität zu steigern.
- Go als Sprache für perfomante, skalierbare Software, die einfach zu warten ist.
- Gründer Robert Griesemer, Rob Pike und Ken Thompson, die bereits an Unix und C gearbeitet haben.

## Fakten über Go:

1. Statische Typisierung: Go ist eine statisch typisierte Sprache, was bedeutet, dass der Typ einer Variable zur Kompilierzeit festgelegt wird. Dies ermöglicht eine bessere Fehlererkennung und Optimierung durch den Compiler.

2. Garbage Collection: Go verfügt über eine eingebaute Garbage Collection, die automatisch den Speicher verwaltet und freigibt, wenn er nicht mehr benötigt wird. Dies erleichtert die Entwicklung von Anwendungen, da Entwickler sich nicht um die manuelle Speicherverwaltung kümmern müssen.

3. Keine Vererbung: Go unterstützt keine Vererbung, sondern verwendet stattdessen Komposition. Dies fördert die Wiederverwendbarkeit von Code und ermöglicht eine flexiblere Gestaltung von Software. Es gibt keine Klassen oder Objekte im traditionellen Sinne, sondern stattdessen werden Strukturen (structs) und Schnittstellen (interfaces) verwendet, um Daten und Verhalten zu organisieren.

4. Concurrency: Go bietet eine starke Unterstützung für Nebenläufigkeit durch Goroutinen und Kanäle. Goroutinen sind leichtgewichtige Threads, die es ermöglichen, mehrere Aufgaben gleichzeitig auszuführen, während Kanäle eine sichere Kommunikation zwischen Goroutinen ermöglichen.

5. Native Ausführung: Go wird direkt zu Maschinencode kompiliert, was zu einer schnellen Ausführung führt. Es gibt keine virtuelle Maschine oder Interpreter, was die Leistung verbessert und die Startzeit von Anwendungen reduziert. Läuft also auf allen Plattformen, auf denen Go unterstützt wird, ohne dass eine zusätzliche Laufzeitumgebung erforderlich ist.
